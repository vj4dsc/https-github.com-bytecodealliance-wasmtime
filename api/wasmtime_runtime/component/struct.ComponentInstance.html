Struct wasmtime_runtime::component::ComponentInstance

source · 
#[repr(C)]
pub struct ComponentInstance { /* private fields */ }
Expand description

Runtime representation of a component instance and all state necessary for the instance itself.

This type never exists by-value, but rather it’s always behind a pointer. The size of the allocation for ComponentInstance includes the trailing VMComponentContext which is variably sized based on the offsets contained within.

pub unsafe fn from_vmctx<R>( vmctx: *mut VMComponentContext, f: impl FnOnce(&mut ComponentInstance) -> R ) -> R

Converts the vmctx provided into a ComponentInstance and runs the provided closure with that instance.

§Unsafety

This is unsafe because vmctx cannot be guaranteed to be a valid pointer and it cannot be proven statically that it’s safe to get a mutable reference at this time to the instance from vmctx.
